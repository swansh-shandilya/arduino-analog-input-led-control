# Potentiometer Controlled LED Indicator

## Description
An Arduino project that uses a potentiometer to control three LEDs based on different voltage ranges.

## Components
- Arduino Uno
- Potentiometer
- 3 LEDs
- Resistors
- Breadboard

## Learning Objectives
- Analog input using `analogRead()`
- Conditional statements (`if-else`)
- LED control
- Basic circuit design
- working of breadboard
